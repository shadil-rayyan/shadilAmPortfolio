# **App Name**: DevFolio

## Core Features:

- Education Section: Displays developer's educational background.
- Skills Section: Showcase developer's skills using logos and short descriptions.
- Project Portfolio: Highlights project entries, linking each to a blog post.
- Tooling & Tech Stack Section: Dedicated section showing programming languages, frameworks, and other development related tooling.
- Contact Section: Includes a contact form and links to email and LinkedIn profiles.
- Performance Optimization: Ensures high performance (95+ Lighthouse score), SEO optimization, and accessibility compliance.
- Alt Text Generation: Generates alt text for images used on the site to enhance accessibility and SEO.

## Style Guidelines:

- Background color: Dark gray (#222222) for a sleek, modern look.
- Primary color: Teal (#008080) to provide a professional and clean look.
- Accent color: Light gray (#D3D3D3) for highlighting text and interactive elements.
- Font: 'Inter' (sans-serif) for both headlines and body text to ensure readability and a modern aesthetic.
- Use brand icons for each listed programming language or tool. If no brand icon is available, choose a simple monochromatic geometric icon.
- Employs a clean, grid-based layout for displaying education, skills, and project entries.
- Subtle animations (e.g., smooth scrolling, hover effects) to enhance user experience.
